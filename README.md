###MJV Test
** Contém as seguintes funcionalidades:

 - Lista de Produtos(Vitrine)
 - Layout responsivo
 - Menu de categorias
 - Detalhe de produto

## Passos para começar
Clonando o Repositório
git clone https://github.com/vfgi/MJVtest.git

Instalando as Dependências
npm install

Inicializando o Servidor
npm start
